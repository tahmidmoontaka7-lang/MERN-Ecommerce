# MERN Ecommerce Project

A full-stack ecommerce web application built using the MERN Stack.

## Features

* User Registration
* User Login Authentication
* Protected Routes
* Product Listing
* Shopping Cart
* Persistent Cart using Local Storage
* Checkout System
* Order Placement
* Order History
* MongoDB Database Integration

## Tech Stack

### Frontend

* React.js
* React Router
* Axios
* Tailwind CSS

### Backend

* Node.js
* Express.js
* MongoDB
* Mongoose
* JWT Authentication
* Bcrypt

## Installation

### Backend

```bash
cd server
npm install
npm run dev
```

### Frontend

```bash
cd client
npm install
npm start
```
